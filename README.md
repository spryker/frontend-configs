> Monorepo for all Frontend configurations for formatters, linters, etc.

| [Stylelint](https://stylelint.io/)          | `@spryker/frontend-config.stylelint` |
| [Tslint](https://palantir.github.io/tslint) | `@spryker/frontend-config.tslint`    |
|                                             |                                      |
